(static) EPSILON

Source:
  • index.js, line 104
It represents the value of 5e-16, which is the smallest number considered as non-zero in this library. In the other words, any number less than Complex.EPSILON is considered as 0. Note that Complex.EPSILON is number instead of instance of Complex.

(static) NaN

Source:
  • index.js, line 76
It represents NaN in this library. It is equivalent to new Complex(NaN). It is important to know that this library does not introduce the concept of Complex Infinity, all Infinity in this library are represented by Complex.NaN.
